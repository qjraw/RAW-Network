## WEEKLY KEYWORD SOURCING RITUAL

Every Monday, spend 15 minutes:

### 1. IG Search Auto-Suggest
- Type each core keyword into Instagram search
- Screenshot the suggestions
- Add new relevant phrases to long-tail list

### 2. Blue Topic Tags
- Find 3–5 viral posts in your niche
- Note the blue topic tags Instagram assigned
- Align your language to match

### 3. TikTok Creator Search Insights
- Check TikTok's search analytics for gap keywords
- Cross-reference with your core list
- Steal phrasing that IG hasn't saturated yet

### 4. Pinterest Bubbles
- Search your core keywords on Pinterest
- Note the suggestion bubbles (pain-based phrases)
- Port the language back to IG captions

## REALITY CHECK

> If you search your core keyword on Instagram and you don't show up — IG doesn't understand you yet.

**Fix:**
- Tighten language (use the exact phrases, not clever synonyms)
- Repeat keywords across ALL elements (caption, alt text, comments, spoken, on-screen)
- Don't post more. Post more precisely.
- Check weekly: search each core keyword, track when you start appearing
