## How to Use This File

1. **During discuss phase:** Add functional and non-functional requirements
2. **During plan phase:** Planner agent checks plan tasks against requirements here
3. **During verify phase:** Each requirement is tested for observable completion
4. **On milestone completion:** Archive completed requirements, scope next milestone
